This code cycles through data files from a TLS and extracts the data
and the parameters corresponding to a sphere. The nominal radius of
the scanned spheres is 50 mm. Data's unit is meters.

-Requires optimization toolbox for MATLAB, 'optim' package for Octave
-Run demo1.m file in MATLAB/Octave

-Requires NumPy, SciPy, MatPlotLib for Python.
-Run demo1.py file in Python
